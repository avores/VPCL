Cross-Modal Contrastive Learning for Visual-Aware Prosody Modeling in Automatic Voice Over


Anonymous submission to INTERSPEECH 2024

Abstract

As an extension of Text-to-Speech (TTS) techniques, Automatic Voice Over (AVO) aims to generate speech that matches with the input text and provided silent video clip. Given the intricacies of aligning speech prosody with visual expressions, exploring cross-modal interactions can achieve more nuanced expressiveness modeling. In this study, we introduce a novel cross-modal contrastive learning method to learn the correlation between facial expression and speech prosody. This approach captures the complex interplay between visual cues and auditory prosody. We evaluate our learned prosody-informed visual features through a cross-modal retrieval task, proving the ability of our approach to capture visual-prosody correlation. Furthermore, we demonstrate that learned visual features can serve as effective cues for visual-aware prosody modeling in AVO.

Fig. 3: Model architecture of visual-prosody enhanced DSU-AVO. Trainable DSU-AVO modules are denoted with green color. Unit Vocoder is pretrained and frozen following [1]. Visual Conditioning module denoted with yellow color is also frozen during DSU-AVO training.

Frameworks to compare
DSU-AVO [1]: an AVO baseline without any visual-aware prosody conditioning.
V-DSU-AVO: we condition DSU-AVO training on visual feature not linked with speech prosody. Specifically, we average the frame-wise facial expression features extracted by pretrained EfficientNet-B2 along temporal dimension to obtain utterance-level visual conditioning (i.e. the Visual Conditioning module in Fig. 3 is a pretrained EfficientNet-B2 followed by a temporal pooling layer and a linear projection layer to align feature dimension).
VP-DSU-AVO: we condition DSU-AVO training on VPCL-learned utterance-level prosody-informed visual feature (i.e. the Visual Conditioning module is visual encoder and projection layer from Fig. 2, trained in VPCL).

For fair comparison, all frameworks share the same DSU-AVO architecture (in green), Unit Vocoder (in red), and learning objective of speech unit prediction [1]. Please refer to [1] for more implementation details. Source code will be released upon acceptance.
